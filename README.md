# Calculadora de Comisiones

Esta es una aplicación web interactiva desarrollada en React, diseñada para calcular comisiones, diferencias y montos transferidos en múltiples monedas. Fue creada específicamente para optimizar tareas relacionadas con mi trabajo, ofreciendo una solución práctica y funcional.

## Características:
- **Cálculo de comisiones**: Calcula las comisiones aplicadas por bancos venezolanos de manera precisa.
- **Cálculo de diferencias**: Determina el porcentaje de diferencia entre dos montos de forma eficiente.
- **Soporte para múltiples monedas**: Compatible con ARS, COP, PEN, VES y ESP (un módulo adaptable a cualquier moneda; "ESP" se refiere a "especial").
- **Interfaz intuitiva**: Diseño pensado para garantizar una experiencia de usuario fluida y amigable.

# Commission Calculator

This is an interactive web application built with React, designed to calculate commissions, differences, and transferred amounts in multiple currencies. It was specifically created to optimize work-related tasks, offering a practical and functional solution.

## Features:
- **Commission calculation**: Accurately calculates the commissions applied by Venezuelan banks.
- **Difference calculation**: Determines the percentage difference between two amounts efficiently.
- **Multi-currency support**: Compatible with ARS, COP, PEN, VES, and ESP (a module adaptable to any currency; “ESP” refers to a -special/custom value).
- **Intuitive interface**: Designed to ensure a smooth and user-friendly experience.
